# Lessons

Each subfolder corresponds to one lesson tag. The folder exists at
`lesson-N` and **all subsequent tags** — it does not disappear. Use this to
look back at earlier narrative without checking out an older commit.

Convention:

```
lessons/NN-slug/
├── README.md       why + what + trade-offs (read first)
├── exercises.md    3 tasks (easy / medium / hard)
└── solution/       reference solution branch checkout
```

At `lesson-00` (this commit) only the framing exists. Lesson 01 onward will
populate this folder.
